# Augmented Labs

## Steps to Run the Website Locally

1. **Clone the Repository**
   ```bash
   git clone https://github.com/AugmentedDevelopers/main-website

2. **Move to the directory**
   ```bash
   cd main-website

3. **Install packages**
   ```bash
   npm install

4. **Run server locally in dev mode**
   ```bash
   npm run dev
